## 📋 完整測試步驟

### 1️⃣ 測試登入功能

#### A. 正確登入
```
1. 訪問：http://localhost:8080/login
2. 輸入帳號：admin
3. 輸入密碼：admin123

```

#### B. 測試錯誤密碼
```
1. 訪問：http://localhost:8080/login
2. 輸入帳號：admin
3. 輸入密碼：wrongpassword
4. 點擊「登入系統」

6. ✓ 訊息內容：「帳號或密碼錯誤，請檢查後重試」

```
1. 訪問：http://localhost:8080/login
2. 不輸入任何內容
3. 點擊「登入系統」
4. ✓ 立即看到錯誤訊息：「請填寫帳號和密碼」
5. ✓ 不會發送請求到伺服器

```
1. 在密碼框輸入密碼
2. 點擊右側的 👁️ 圖示

4. ✓ 圖示變為 🙈
5. 再次點擊
6. ✓ 密碼恢復隱藏
```

## 📝 登入表單欄位說明

### username 欄位
- **名稱**: `username`
- **類型**: text
- **必填**: ✓
- **驗證**: 不可為空
- **自動完成**: username

### password 欄位
- **名稱**: `password`
- **類型**: password
- **必填**: ✓
- **驗證**: 不可為空
- **自動完成**: current-password
- **特殊功能**: 可切換顯示/隱藏

### remember-me 欄位
- **名稱**: `remember-me`
- **類型**: checkbox
- **必填**: ✗
- **功能**: 記住登入狀態

### CSRF Token
- **自動包含**: ✓
- **由 Thymeleaf 自動生成**

## 🔧 如果遇到問題

### 問題 1: 登入頁面無法訪問
```bash
# 檢查應用是否運行
ps aux | grep java

# 重新啟動
./mvnw spring-boot:run -Dspring-boot.run.profiles=dev
```

### 問題 2: 登入後沒有跳轉
- 檢查瀏覽器控制台錯誤
- 確認 JavaScript 已啟用
- 清除瀏覽器 Cookie 和快取

### 問題 3: 錯誤訊息不顯示
- 確認 Thymeleaf 模板正確載入
- 檢查瀏覽器開發者工具
- 查看應用日誌
